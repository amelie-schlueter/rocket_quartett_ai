## How it works

1. Clone the Repo
2. Create a .env.local file in your root directory

```
OPENAI_API_KEY=YOUR_API_KEY
```

3. Install all the dependecies using pnpm install

```
pnpm install
```

4. Start the Development Sever with pnpm run dev

```
pnpm run dev
```
